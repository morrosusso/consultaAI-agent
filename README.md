# ConsultAI Agent

AI-powered consultancy agent with Gmail integration, built with Next.js and Claude AI.

## Features
- 📨 Read real Gmail inbox
- 🧠 AI-drafted replies powered by Claude
- 📨 Send replies directly from the app
- 💬 AI chat assistant

## Setup

### Environment Variables
Add these in Vercel (Settings → Environment Variables):

```
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
NEXTAUTH_URL=https://your-vercel-url.vercel.app
NEXTAUTH_SECRET=any_random_32_character_string
ANTHROPIC_API_KEY=your_anthropic_api_key
```

### Deploy
1. Push this repo to GitHub
2. Import in Vercel
3. Add environment variables
4. Deploy!
